# ClawLess Playground

Try [ClawLess](https://github.com/open-gitagent/clawless) in your browser — a serverless runtime for Claw AI Agents powered by WebContainers.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fopen-gitagent%2Fclawless-playground)

## Run Locally

```bash
git clone https://github.com/open-gitagent/clawless-playground.git
cd clawless-playground
npm install
npm run dev
```

## What is ClawLess?

ClawLess runs AI agents entirely in the browser — no server required. It provides a full sandboxed Node.js environment via WebContainers (WASM) with a built-in editor, terminal, policy engine, and audit logging.

Learn more at [github.com/open-gitagent/clawless](https://github.com/open-gitagent/clawless).
